# ZdorovLife Telegram Mini App

Готовый учебный проект: Telegram-бот + встроенное мини-приложение по теме здорового образа жизни и медицинской профилактики.

## Что это такое
Это не просто чат с кнопками, а **Telegram Mini App**:
- открывается как отдельное приложение внутри Telegram;
- имеет красивый игровой интерфейс;
- показывает показатели здоровья;
- обучает через ситуации выбора и профилактические комментарии.

## Что внутри
- `bot.py` — Telegram-бот на aiogram 3;
- `server.py` — FastAPI-сервер для выдачи мини-приложения;
- `web/index.html` — интерфейс мини-приложения;
- `web/styles.css` — стиль;
- `web/app.js` — игровая логика.

## Как запустить локально
1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создай файл `.env` на основе `.env.example`.
3. Запусти сервер:
   ```bash
   uvicorn server:app --host 0.0.0.0 --port 8000
   ```
4. Сделай публичный URL через ngrok / cloudflared / deploy.
5. В `.env` укажи публичный URL как `WEBAPP_URL`.
6. Запусти бота:
   ```bash
   python bot.py
   ```
